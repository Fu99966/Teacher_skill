# Worksheet Generator

## 输入

- 学科
- 年级
- 知识点
- 难度
- 题型
- 题量

## 输出

- 学生版题目
- 教师版答案
- 解析
- 知识点覆盖说明

## 题型

- 选择题
- 填空题
- 判断题
- 简答题
- 应用题
- 阅读理解题

## 原则

题目要贴合年级水平，答案准确，解析简明。需要填入模板时，输出 JSON 字段，不直接控制 Word 样式。
